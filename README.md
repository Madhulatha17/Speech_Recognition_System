# DESCRIPTION OF TASK

The objective of this task is to develop a Speech Recognition System using Python. The program takes an audio file as input and converts the spoken words into text using the SpeechRecognition library. The recognized text is displayed on the screen and saved into an output text file (`output.txt`) for future reference.

The implementation involves loading an audio file, processing the speech data, converting speech to text using Google's Speech Recognition API, and storing the resulting text in a file. This task demonstrates the practical application of speech processing and natural language technologies in automating audio-to-text conversion.

Tools and Technologies Used:

* Python
* SpeechRecognition Library
* Visual Studio Code (VS Code)
* WAV Audio File Format

The program successfully recognizes speech from the audio file and saves the converted text into `output.txt`.
